# 0.事前準備

## Nodeが入っていること
    
- 以下のコマンドでバージョンが表示されること
- Nodeのバージョンは12以上推奨
    - 10以上ならたぶん動く

```sh
node -v
npm -v
```

- 入っていなければ公式からダウンロードしてください
    - [https://nodejs.org/ja/](https://nodejs.org/ja/)

## yarnが入っていること

- 以下のコマンドでバージョンが表示されること
- npmでもいいですが手順はyarnを使ってます

```sh
yarn -v
```

- 入っていなければ以下のコマンドでインストールしてください

```sh
npm i -g yarn
```
